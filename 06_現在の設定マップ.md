## Program 1 - Left FN レイヤー (Level 1)

Left Fn を押しながら各キーを押した時の動作:

```
+------+----+----+----+----+----+----+------+----+----+-----+-----+------+----+----+
| 特殊 | F1 | F2 | F3 | F4 | F5 | F6 | 特殊 | F8 | F9 | F10 | F11 | 特殊 | -- | -- |
|  150 |  2 |  3 |  4 |  5 |  6 |  7 |  149 |  9 | 10 |  11 |  12 |  159 |    |    |
+------+----+----+----+----+----+----+------+----+----+-----+-----+------+----+----+
| MsDly| MsSpd| MsUp| MsPx| -- | -- | -- | -- | Ins | Home| PgUp| RepRt| ResDly| 特殊| -- |
|  173 |  158 |  128 |  157 |    |    |    |    |  78 |  79 |  80 |  174 |  175  | 179 |    |
+------+------+------+------+----+----+----+----+-----+-----+-----+------+-------+-----+----+
| MsL  | MsDn | MsR  | MsFn | -- | -- | -- | Del | End | PgDn| --  | --   |
| 126  |  129 |  127 |  133 |    |    |    |  81 |  82 |  83 |     |      |
+------+------+------+------+----+----+----+-----+-----+-----+-----+------+-----+
| 特殊 | MsLC | MsWC | MsRC | MsFn| -- | -- | 特殊| PrSc| ScrLk| Pause| 特殊| 特殊|
|  198 |  130 |  132 |  131 |  134 |   |    | 207 |  84 |  85  |  86  | 151 | 145 |
+------+------+------+------+------+---+----+-----+-----+------+------+-----+-----+
| 特殊 | LFN  | LFN  | LFN  | --  | 特殊| --  | -- | -- | -- | RFN |
|  152 |  166 |  166 |  166 |     | 156 |     |    |    |    | 144 |
+------+------+------+------+-----+-----+-----+----+----+----+-----+
```

**凡例**:
- `--` = 未割当 (Mode=0, HWCode=0)
- `MsUp/Dn/L/R` = マウスカーソル移動
- `MsLC/RC/WC` = マウス左/右/ホイールクリック
- `MsSpd/Px/Dly` = マウス速度/ピクセル/遅延調整
- `RepRt/ResDly` = キーリピート速度/レスポンス遅延
- 数字 = HWCode 値
