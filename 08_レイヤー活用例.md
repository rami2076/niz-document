# レイヤー活用例集

ATOM66-1M の 3つの Program モード × 3レイヤーを活かした構成例。

---

## 構成例A: プログラマー向け

### Program 1: 通常作業用

**Base レイヤー**
- 標準 QWERTY 配列
- CapsLock → Left Ctrl に変更（プログラミングで多用するため）

**Left FN レイヤー**
```
数字行:     F1〜F12
H/J/K/L:   ← ↓ ↑ → （Vim風ナビゲーション）
U:          Home
I:          Page Up
O:          Page Down
P:          End
N:          Delete
M:          Backspace
[:          キーリピート速度
]:          レスポンス遅延
```

**Right FN レイヤー**
```
数字行:     F1〜F12（Left FNと同じ）
WASD:       マウス移動
ZXC:        マウスクリック
,./         Print Screen / Scroll Lock / Pause
```

## 構成例B: 用途別 Program 切り替え

### Program 1: 日常作業・文書作成

**Base**: 標準配列
**Left FN**: F1-F12 + 矢印 + メディアコントロール
**Right FN**: マウスエミュレーション

### Program 2: ゲーミング

**Base**: 標準配列（変更なし）
**Left FN**:
```
1〜5:       アイテムスロット（F1〜F5）
Q:          スキル1（F6）
E:          スキル2（F7）
R:          スキル3（F8）
Tab:        スコアボード（F9）
```
**Right FN**: マクロキー（ゲーム固有の操作）

### Program 3: Mac 用

**Base**:
```
Left Alt  → Left Super (Command)
Left Super → Left Alt (Option)
※ Mac のショートカットに合わせた配置
```
**Left FN**: 標準的な F キー + ナビゲーション
**Right FN**: メディアコントロール + 画面輝度

## 構成例C: ホームポジション重視

全ての操作をホームポジション（ASDF / JKL;）から手を離さずに行う構成。

### Base レイヤー
- 標準 QWERTY
- CapsLock → LFN に変更（左手親指や小指で押しやすい位置）

### Left FN レイヤー（CapsLock 長押しで発動）
```
右手ホーム行:
  H → Left Arrow      ←
  J → Down Arrow      ↓
  K → Up Arrow        ↑
  L → Right Arrow     →
  ; → End
  ' → Home

右手上段:
  Y → Page Up
  U → Home
  I → Up Arrow
  O → Page Down
  P → End

右手下段:
  N → Delete
  M → Backspace
  , → Insert

左手:
  Q → ESC
  1〜= → F1〜F12
```

### Right FN レイヤー
```
右手ホーム行:
  J → Volume Down
  K → Mute
  L → Volume Up
  ; → Play/Pause

WASD:
  マウスカーソル操作

ZXC:
  マウスクリック操作
```

## 構成例D: 最小限カスタム（初心者向け）

ほぼデフォルトのまま、不便な部分だけ変更する構成。

### 変更点のみ（Base レイヤー）
```
変更1: CapsLock(ID 61) → Left Ctrl
  理由: CapsLockは誤爆しやすく、Ctrlの方が使用頻度が高い

変更2: Right Shift(ID 54) を RShift のままにする
  注意: ATOM66 の Right Shift はデフォルトで ↑ と兼用モードになっている場合あり
        Fn + Right Shift で動作モード切替可能（↑のみ / Shiftのみ / 両方）
```

### Left FN レイヤー: デフォルトのまま
```
数字行 → F1-F12
I/O/P → Insert/Home/PageUp
K/L/; → Delete/End/PageDown
WASD → マウス
```

## カスタマイズのコツ

1. **まず1つだけ変更して試す**: 一度に大量変更すると混乱する
2. **変更前に Save**: 必ず現在の設定を .pro ファイルに保存してから変更
3. **DEFAULT.pro を残す**: 工場出荷状態に戻せるように保管しておく
4. **Left FN と Right FN を役割分担**: 
   - Left FN = ナビゲーション・F キー（左手で FN、右手で操作）
   - Right FN = マウス・メディア（右手で FN、左手で操作）
5. **Program モードを使い分ける**: 
   - 毎日使うメイン設定は Program 1
   - 特定用途は Program 2/3 に入れて切り替え
