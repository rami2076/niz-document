# NiZ ATOM66-1M キーボードショートカット一覧

## モード・レイアウト切り替え

| ショートカット | 操作 | 長押し |
|---------------|------|--------|
| Right Fn + = (Mキー) | Office ↔ Program モード切り替え | 2秒 |
| Fn + Left Alt | Windows ↔ Mac キーレイアウト切り替え | 3秒 |
| Right Fn + - (マイナス) | 有線 ↔ Bluetooth 切り替え | - |

## Bluetooth 操作

| ショートカット | 操作 | 長押し |
|---------------|------|--------|
| Right Fn + 8 | Bluetooth デバイス1に接続 | - |
| Right Fn + 9 | Bluetooth デバイス2に接続 | - |
| Right Fn + 0 | Bluetooth デバイス3に接続 | - |
| Right Fn + 8 | Bluetooth デバイス1とペアリング | 3秒 |
| Right Fn + 9 | Bluetooth デバイス2とペアリング | 3秒 |
| Right Fn + 0 | Bluetooth デバイス3とペアリング | 3秒 |

## FN レイヤーロック

| ショートカット | 操作 | 長押し |
|---------------|------|--------|
| Left Fn + Right Fn（左を先に） | Left FN レイヤーをロック（LED 2回点滅） | 3秒 |
| Right Fn + Left Fn（右を先に） | Right FN レイヤーをロック（LED 3回点滅） | 3秒 |
| Left Fn + Right Fn（再度） | FN ロック解除（LED 1回点滅） | 3秒 |

## 入力調整

| ショートカット | 操作 |
|---------------|------|
| Fn + ] | キーレスポンス遅延切り替え（500ms → 300ms → 100ms） |
| Fn + [ | キーリピート速度切り替え（x1 → x2 → x4 → x8） |
| Right Fn + 7 | APC（打鍵深度）切り替え（2mm / 2.5mm / 3mm） |

## マウスエミュレーション（FN レイヤーのデフォルト設定）

| ショートカット | 操作 |
|---------------|------|
| Fn + W | マウスカーソル上移動 |
| Fn + A | マウスカーソル左移動 |
| Fn + S | マウスカーソル下移動 |
| Fn + D | マウスカーソル右移動 |
| Fn + Z | マウス左クリック |
| Fn + X | マウスホイールクリック |
| Fn + C | マウス右クリック |
| Fn + Q | マウスカーソル速度調整 |
| Fn + E | マウスピクセル調整 |

## リセット

| ショートカット | 操作 | 長押し |
|---------------|------|--------|
| ESC + Left Ctrl + Right Ctrl + F12 | ファクトリーリセット（全設定を初期化） | 5秒 |
