# NiZ ATOM66-1M キーボード レイヤー設定 調査レポート

調査日: 2026-04-19

---

## 1. キーボードのモード構成

NiZ ATOM66-1M には以下の **4つのモード** がある。

| モード | 説明 | カスタマイズ可否 |
|--------|------|-----------------|
| **Office モード** | 工場出荷時のデフォルト設定。固定で変更不可 | 不可 |
| **Program 1** | カスタム設定1。専用ソフトで自由に設定可能 | 可能 |
| **Program 2** | カスタム設定2。Program 1とは独立した設定 | 可能 |
| **Program 3** | カスタム設定3。Program 1/2とは独立した設定 | 可能 |

### モード切り替え方法

- **Office → Program モードの切り替え**: `Right Fn` + `=`(Mマーク付きキー）を **2秒間長押し**
  - Mキーのインジケーターが点灯 → Program モードに入った
  - もう一度同じ操作で Office モードに戻る
- **Program 1/2/3 の切り替え**: ソフトウェア上で対象のプログラムモードを選択して設定を書き込む
  - 82ECなどの上位モデルではMキーの点灯パターン（常時点灯=Program1、点滅=Program2）で判別可能
  - ATOM66-1Mでは、Mキーの点灯/点滅回数で判別する仕組みがある模様

> **重要**: Office モードでは専用ソフトウェアからキーボードへの設定書き込みができない。必ず **Program モードに切り替えてから** 設定を行うこと。

## 2. レイヤー構成（各モード共通）

各モード（Program 1/2/3）には、それぞれ **3つのレイヤー** がある。

| レイヤー | 名称 | 発動条件 |
|----------|------|----------|
| **Base レイヤー（None層）** | 標準レイヤー | キーを単独押し |
| **Left FN レイヤー** | 左Fnレイヤー | Left FN キーを押しながら他のキーを押す |
| **Right FN レイヤー** | 右Fnレイヤー | Right FN キーを押しながら他のキーを押す |

### レイヤーの仕組み

- **Base レイヤー**: 通常のキー入力で使われるレイヤー。アルファベット、数字、記号など
- **Left FN レイヤー**: Left FN キーと組み合わせた時だけ出力されるキーの設定
- **Right FN レイヤー**: Right FN キーと組み合わせた時だけ出力されるキーの設定
- Left FN と Right FN はそれぞれ独立したレイヤーを持ち、異なるキーマッピングを設定可能

## 3. .pro ファイルにおける Level の対応関係

専用ソフトウェアが保存する `.pro` ファイル（XML形式）では、全ての設定が `Level` 属性で管理されている。

| Level | モード | レイヤー |
|-------|--------|----------|
| **Level 0** | Program 1 | Base（標準） |
| **Level 1** | Program 1 | Left FN |
| **Level 2** | Program 1 | Right FN |
| **Level 3** | Program 2 | Base（標準） |
| **Level 4** | Program 2 | Left FN |
| **Level 5** | Program 2 | Right FN |
| **Level 6** | Program 3 | Base（標準） |
| **Level 7** | Program 3 | Left FN |
| **Level 8** | Program 3 | Right FN |

合計: 3モード x 3レイヤー = **9 Level（Level 0〜8）**

## 4. レイヤーを分ける（使い分ける）ための手順

### 4.1 準備

1. キーボードをUSB Type-Cケーブルでパソコンに接続する
2. キーボードを **Program モード** に切り替える
   - `Right Fn` + `=`（Mキー）を2秒間長押し → Mキーのインジケーターが点灯
3. 専用ソフトウェア `66EC(XRGB)Ble.exe` を起動する

### 4.2 現在の設定を読み込む

1. ソフトウェアが起動したら、右上の **「English」** を押して英語表示に切り替え（中国語で表示される場合）
2. **「ReadAll」ボタン** をクリックして、キーボードの現在の設定を読み込む

> **最重要**: 設定変更前に必ず ReadAll を行うこと。これを省略すると既存の設定が全てリセットされる危険がある。

### 4.3 レイヤーの設定変更

1. ソフトウェア画面上でキーボードのビジュアルレイアウトが表示される
2. 画面上部や左側に **レイヤー選択** の項目がある
   - 「None」(Base)、「Left FN」、「Right FN」の各レイヤーをタブやドロップダウンで切り替え可能
3. 変更したいレイヤーを選択した状態で：
   - 変更したいキーをクリック
   - 「KeySettings」パネルで以下を選択:
     - **Single**: 単一キーの割り当て
     - **Combo**: 最大6キーの同時押しを割り当て
     - **Macro**: マクロ（連続キー操作）を割り当て
     - **Function**: LFN（左Fn）やRFN（右Fn）を割り当て
4. 「Record」をクリックしてキーを記録

### 4.4 FN キーの割り当て方法

FN キー自体の位置を変更したい場合：

1. 割り当てたいキーをクリック
2. **「Function」タブ** を開く
3. 左Fnキーには **「LFN」** を、右Fnキーには **「RFN」** を選択して割り当て

### 4.5 設定の書き込み

1. 全ての変更が完了したら **「WriteKeys」ボタン** をクリック
2. 設定がキーボード本体に書き込まれる
3. 書き込みが完了すると設定が即座に反映される

### 4.6 設定ファイルの保存

- **「Save」ボタン**: 現在の設定を `.pro` ファイルとして PC に保存
- **「Open」ボタン**: 保存した `.pro` ファイルを読み込む

## 5. FN レイヤーのロック機能

Left FN と Right FN を **同時に3秒間長押し** すると、FN レイヤーがロック（常時有効化）される。

- 先に押した FN キーのレイヤーがロックされる
  - Left FN を先に押す → Left FN レイヤーがロック
  - Right FN を先に押す → Right FN レイヤーがロック
- ロック状態のインジケーター:
  - LEDが2回点滅 → Left FN ロック
  - LEDが3回点滅 → Right FN ロック
  - LEDが1回点滅 → ロック解除

## 6. よくある注意点

- **Officeモードでは設定変更が反映されない**: 必ずProgramモードに切り替えてから操作する
- **ReadAllを忘れない**: 設定変更前にReadAllしないと既存設定が失われる可能性がある
- **Windows環境が必要**: 専用ソフトウェアはWindows専用。MacユーザーはParallelsやBoot Campを使用
- **Mac/Win切り替え**: `Fn` + `Left Alt` を3秒長押しで、Windowsレイアウト ↔ Macレイアウトを切り替え。ソフトウェア使用前にWindowsレイアウトにしておくこと
- **ファクトリーリセット**: `ESC` + `Left Ctrl` + `Right Ctrl` + `F12` を5秒長押し
