# 設定変更の具体的手順（ステップバイステップ）

---

## 前提: 毎回必ず行うこと

```
1. USB ケーブルでキーボードを接続
2. Right Fn + = (Mキー) を2秒長押し → Program モードにする（Mインジケーター点灯）
3. 66EC(XRGB)Ble.exe を管理者権限で起動
4. 「ReadAll」を必ずクリック ← これを忘れると既存設定が消える
```

---

## 例1: 単一キーの変更（CapsLock を Ctrl にする）

**目的**: CapsLock（ID 61）を Left Ctrl として使いたい

```
手順:
1. レイヤー選択 → 「None」（Base レイヤー）を選択
2. 画面上の CapsLock キー（ID 61）をクリック
3. KeySettings パネルで「Single」を選択
4. 「Record」をクリック
5. キーボードの Left Ctrl キーを押す
6. 「Stop」をクリック
7. 画面上で CapsLock の表示が「L-Ctrl」に変わったことを確認
8. 「WriteKeys」をクリック → キーボードに書き込み
9. 「Save」で .pro ファイルとしてバックアップ保存
```

---

## 例2: FN レイヤーにキーを追加（Fn + H = Left Arrow）

**目的**: Left Fn + H で ← (左矢印) を入力したい

```
手順:
1. レイヤー選択 → 「Left FN」を選択
2. 画面上の H キー（ID 36）をクリック
3. KeySettings パネルで「Single」を選択
4. 「Record」をクリック
5. キーボードの ← (Left Arrow) キーを押す
6. 「Stop」をクリック
7. 「WriteKeys」をクリック → 書き込み
```

**応用**: 同様に Vim 風のナビゲーションを設定できる
```
Left Fn + H → Left Arrow
Left Fn + J → Down Arrow
Left Fn + K → Up Arrow
Left Fn + L → Right Arrow
```

---

## 例3: コンボキーの設定（1キーで Ctrl+C）

**目的**: Left Fn + C で Ctrl+C（コピー）を1キーで実行したい

```
手順:
1. レイヤー選択 → 「Left FN」を選択
2. 画面上の C キー（ID 46）をクリック
3. KeySettings パネルで「Combo」を選択
4. 「Record」をクリック
5. Ctrl キーを押しながら C キーを押す（同時押し）
6. 「Stop」をクリック
7. 「WriteKeys」をクリック
```

**注意**: コンボキーは最大6キーまで同時登録可能

---

## 例4: FN キーの位置を変更する

**目的**: CapsLock の位置を Left Fn にしたい

```
手順:
1. レイヤー選択 → 「None」（Base レイヤー）を選択
2. 画面上の CapsLock キー（ID 61）をクリック
3. KeySettings パネルで「Function」タブを選択
4. 「LFN」を選択
5. 「WriteKeys」をクリック

※ 元の Left Fn キー（ID 59）に別のキーを割り当てるのを忘れずに
```

---

## 例5: Right FN レイヤーだけ別の設定にする

**目的**: Left Fn は既存のまま、Right Fn に独自設定を入れたい

```
手順:
1. レイヤー選択 → 「Right FN」を選択
2. ここで設定した内容は Right Fn を押しながら入力した時だけ有効
3. Left FN レイヤーとは完全に独立しているので自由に設定可能
4. 例:
   - Right Fn + 数字行 → メディアコントロール（音量、再生等）
   - Right Fn + WASD → 矢印キー
   - Right Fn + その他 → よく使うショートカット
5. 各キーを設定後「WriteKeys」で書き込み
```

## 設定を元に戻す方法

### 特定のキーだけ戻す
```
1. ReadAll でキーボードから読み込み
2. 該当キーをクリック
3. 「Clear」をクリック → そのキーの設定が消去される
4. WriteKeys で書き込み
```

### 全設定を工場出荷状態に戻す
```
方法A: キーボード本体で
  ESC + Left Ctrl + Right Ctrl + F12 を5秒長押し

方法B: DEFAULT.pro ファイルから復元
  1. ソフトウェアで「Open」をクリック
  2. DEFAULT.pro を選択して読み込み
  3. 「WriteKeys」で書き込み
```
