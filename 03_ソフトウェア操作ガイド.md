# 66EC(XRGB)Ble.exe 操作ガイド

## 概要

`66EC(XRGB)Ble.exe` は NiZ ATOM66-1M キーボード専用のキーマッピング設定ソフトウェア。
Windows 専用（Mac では Parallels Desktop や Boot Camp 経由で使用可能）。

ソフトウェアの場所:
```
専用ソフトウェア-ATOM66-1M\66EC(XRGB)Ble.exe
```

---

## 使用前の準備

### 1. キーボード接続
- USB Type-C ケーブルで PC に有線接続する
- **ソフトウェアは有線接続でのみ動作する**（Bluetooth 接続では不可）

### 2. Windows キーレイアウトに切り替え
- `Fn` + `Left Alt` を3秒間長押し → Windows レイアウトに切り替え
- Mac レイアウトだとソフトウェアが正常に通信できない場合がある

### 3. Program モードに切り替え
- `Right Fn` + `=`（Mマーク付きキー）を2秒間長押し
- Mキーのインジケーターが点灯したら Program モード

### 4. ソフトウェアを管理者権限で起動
- `66EC(XRGB)Ble.exe` を右クリック →「管理者として実行」
- 起動後、中国語表示の場合は右上の **「English」** をクリックして英語に切り替え

## 基本操作フロー

### 設定の読み込み（最初に必ず実行）

```
1. ReadAll をクリック
   → キーボード本体から現在の全設定を読み込む
   → これを省略すると設定が初期化される危険あり
```

### ファイルから読み込む場合

```
1. Open をクリック
2. .pro ファイルを選択
   → 保存済みの設定ファイルを読み込む
```

---

## レイヤー別の設定変更手順

### Base レイヤー（None）の編集

```
1. レイヤー選択で「None」を選択
2. 変更したいキーを画面上でクリック
3. KeySettings パネルで設定:
   - Single: 単一キーの割り当て
     → Record をクリック → 割り当てたいキーを押す → Stop
   - Combo: 複数キーの同時押し（最大6キー）
     → Record をクリック → 割り当てたい組み合わせを押す → Stop
```

### Left FN レイヤーの編集

```
1. レイヤー選択で「Left FN」を選択
2. 変更したいキーをクリック
3. KeySettings で割り当てを設定
   → この設定は Left Fn キーを押しながら当該キーを押した時に発動する
```

### Right FN レイヤーの編集

```
1. レイヤー選択で「Right FN」を選択
2. 変更したいキーをクリック
3. KeySettings で割り当てを設定
   → この設定は Right Fn キーを押しながら当該キーを押した時に発動する
```

---

## FN キー自体の割り当て変更

FN キーの物理位置を別のキーに移動したい場合:

```
1. FN を割り当てたいキーをクリック
2. KeySettings の「Function」タブを開く
3. Left Fn にしたい場合は「LFN」を選択
4. Right Fn にしたい場合は「RFN」を選択
5. 確定
```

---

## 設定の書き込み

```
1. 全ての変更が完了したら「WriteKeys」をクリック
2. キーボード本体に設定が書き込まれる
3. 書き込み完了後、設定は即時反映される
4. 設定はキーボード本体のメモリに保存されるため、
   他のPC/Mac/タブレットに接続しても同じ設定で動作する
```

---

## 設定ファイルの管理

```
Save: 現在の設定を .pro ファイルとして PC に保存
Open: 保存した .pro ファイルを読み込む
```

→ 複数の設定パターンを .pro ファイルとして保管しておけば、
   用途に応じた切り替えが容易になる

---

## Program 1/2/3 の使い分け

ATOM66-1M は 3つの Program モードを持つ。
ソフトウェア上でどの Program に書き込むかを選択できる（モデルによりUI差異あり）。

### 活用例
- **Program 1**: 通常のプログラミング用レイアウト
- **Program 2**: ゲーミング用（WASDを矢印キーに変更等）
- **Program 3**: Mac 用に最適化したレイアウト

## トラブルシューティング

| 問題 | 解決方法 |
|------|----------|
| ソフトウェアがキーボードを認識しない | 有線接続を確認。Windowsレイアウトに切り替え。管理者権限で実行 |
| 設定が反映されない | Program モードに切り替えてからWriteKeys。Office モードでは反映されない |
| 既存の設定が消えた | WriteKeys前にReadAllを忘れた可能性。DEFAULT.pro から復元可能 |
| ソフトウェアが中国語表示 | 右上の「English」ボタンをクリック |
| Mac で使えない | Parallels Desktop でWindows VMを用意し、USBパススルーで接続 |
| Right Shift が効かない | 右Shiftのモード切替：`Fn` + `Right Shift` で、↑ / Shift / 両方 を切り替え |
